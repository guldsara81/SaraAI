# # SaraAI

**Version:** 0.1.0 "Genesis"

SaraAI är ett öppet AI-projekt byggt från grunden med Python.

## Vision

Målet med SaraAI är att skapa en modulär AI-plattform som kan:

- Föra naturliga konversationer
- Lära sig från data
- Ha kort- och långtidsminne
- Använda verktyg
- Köras lokalt eller på egna servrar
- Publiceras som ett open source-projekt
